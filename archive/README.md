# Archived Board-Game Attempts

`archive/attempts/` contains historical board-game and FairGame implementation
attempts. They are kept locally as reference material for the rebuild, but they are
ignored by the top-level Git repository.

Do not continue development inside these archived attempts unless the user explicitly
asks for that. The active rebuild should live in `/Volumes/T9/code/2-boards/fairgame-rebuild`.

Useful references from the audit:

- `archive/attempts/fairgame-gpt5`: compact two-board model and scoring ideas.
- `archive/attempts/board2`: useful chess-rule reference and tests.
- `archive/attempts/chess2`: older double-board chess match model and timer ideas.
- `archive/attempts/claude-fairgame`: timer behavior and PouchDB lessons.
- `archive/attempts/fairgame2`: compositional dual-board abstraction idea.
